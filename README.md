# Princess Peach Mini Figure (E-Design 100)
🏆 Award: Most Creative Design (Peer-voted, ~250–300 students)
## Overview

This project is a CAD-designed mini figure of Princess Peach created for E-Design 100. The assignment focused on designing a small-scale character model under strict size and assembly constraints, combining creativity with technical modeling skills.

This design was awarded **“Most Creative Design”** out of approximately 250–300 student submissions through a peer voting process.

## Project Concept

The goal of this project was to design a recognizable and original mini character using CAD software. I chose Princess Peach as the subject and adapted her features into a simplified, manufacturable form suitable for 3D printing.

The design includes a themed display concept with a mystery box accessory and a star element to enhance the visual appeal.

## Design Requirements

* Maximum size constraints (limited height and width)
* Minimum of **3 original parts**
* At least **2 accessories**
* Must be suitable for 3D printing

## Features

* Multi-part character design (head, bust, and dress)
* Custom-designed crown and mystery box accessories
* Stylized adaptation of Princess Peach
* Designed for small-scale 3D printing

## CAD & Modeling Challenges

One of the main challenges in this project was modeling organic shapes—especially the hair—using CAD software like Fusion 360, which is not optimized for highly fluid or sculpted forms.

To address this, I simplified and stylized certain features while maintaining the recognizable appearance of the character.

## Manufacturing

The model was designed for 3D printing, ensuring that all parts were compatible with additive manufacturing constraints.

## Tools Used

* Fusion 360 (CAD modeling)
* 3D printing workflow

## Course Context

This project was completed as part of E-Design 100, an introductory engineering design course focused on CAD modeling, creativity, and problem-solving under constraints.
